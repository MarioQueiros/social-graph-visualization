WIN-PROLOG Version 4.900
========================

Welcome to the Case Based Reasoning Toolkit for WIN-PROLOG 4.900!
This toolkit requires that you have previously installed WIN-PROLOG and Prodata.
 If you have not already done so please read the information on your 
WIN-PROLOG CD on how to install your WIN-PROLOG system.

Loading and Running the Case Based Reasoning Toolkit 
----------------------------------------------------
To run the Case Based Reasoning Toolkit you first run WIN-PROLOG
version 4.900 and then enter the following command at the WIN-PROLOG
command line:

        ?- ensure_loaded( system(cbr_api) ).
