WebFlex Demos

WebFlex is LPA's Expert System for the Internet. Based on ProWeb Server technology, this toolkit enables you easily to deliver expert system applications over the World Wide Web.

We have assembled a small selection of demonstration programs to give you some examples of what can be down with WebFlex; to explore these demos, please select from the following links:

    Demo Explanation
    Forms A demonstration of the use of forms from WebFlex
    Robbie Robbie helps select items to put into a shopping cart
    Solvents Select the correct solvent to use when cleaning equipment
    Forest Yield Recommend a species of tree seed to maximise forest yield
    Insurance Detecting insurance fraud using Bayes theorem.
    Fuzzy Car Ranking cars through fuzzy logic.
    Pesticide An expert for selecting the appropriate pesticide.
    Timetable Generating time-tables using forward chaining.
    Holidays An implemention of a decision tree.
