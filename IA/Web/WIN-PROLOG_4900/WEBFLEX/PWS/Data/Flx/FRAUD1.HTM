Bayesian Belief Network Example

This example demonstrates the use of forward-chaining uncertainty rules in a motor insurance fraud detection problem.

Description

The example implements a probablistic fraud detection expert system.

The problem is tackled in four distinct stages:

  1. Initialise fraud indicators to their a priori probabilities.
  2. Ask the user for indicator probabilities, either from a list of yes, no and not_known or by means of a number between 0 (false) and 1 (true).
  3. Calculate the overall probability of fraud using Bayesian updating.
  4. Carry out hypothetical reasoning to see what unknown indicators would be worth investigating (if any).
