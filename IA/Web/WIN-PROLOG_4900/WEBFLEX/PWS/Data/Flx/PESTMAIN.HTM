Pesticide Expert System Example

This example recommends a pesticide to apply to a given crop to combat the given pests infesting that crop. The knowledge of the expert system is:
  1. Pests: Aphid, Caterpillar, Red Spider Mite, Slug
  2. Plants: Strawberry, Potato, Tomato, Brussel Sprout
  3. Products: Metaldehide, Malathion, Pyrethrum

    4. The rules are based on:

  4. Malathion kills Aphid, Caterpillar and Red Spider Mite
  5. Pyrethum kills Aphid and Caterpillar
  6. Metaldehide kills Slugs
  7. Malathion kills pests more effectively than Pyrethum, but should not be applied to soft fruit within 14 days of harvest
  8. If there is more than one type of pest, they should be treated with the following priority: (1)Red Spider Mite, (2)Caterpillar, (3)Aphid, (4)Slug
  9. No two chemicals should be used together
